4.Раскомментируйте блок кода, примерно расположенный на строчках 29–42 файла main.tf. Выполните команду terraform validate. Объясните, в чём заключаются намеренно допущенные ошибки. Исправьте их.

### Ответ.

Намеренно допущенные ошибки:

resource "docker_image" { ... }

У ресурса отсутствует имя. В Terraform у ресурса должно быть две метки: тип и имя:

resource "docker_image" "nginx" { ... }.

resource "docker_container" "1nginx" { ... }

Имя ресурса начинается с цифры. Допустимо, чтобы имя начиналось с буквы или подчёркивания.

random_password.random_string_FAKE.resulT
– неправильно указано имя ресурса (random_string_FAKE вместо существующего random_string),
– неправильно указан атрибут (resulT вместо result в нижнем регистре).
